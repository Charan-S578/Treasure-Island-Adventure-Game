# Treasure Island Adventure Game 🏝️💎

A fun text-based adventure game built with Python where players make choices to navigate through Treasure Island in search of hidden treasure. Every decision affects the outcome of the game.

---

## Features

* Interactive story-based gameplay
* Multiple decision paths
* Win and Game Over outcomes
* ASCII art introduction
* User-friendly command-line interface

---

## Technologies Used

* Python 3

---

## Game Story

You are stranded on Treasure Island with one mission:

🎯 **Find the hidden treasure!**

Players must make the correct choices to survive challenges and reach the treasure safely.

---

## Game Flow

1. Choose a direction at the crossroad.
2. Decide whether to wait or swim across the lake.
3. Select the correct colored door.
4. Reach the treasure and win the game.
